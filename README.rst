About
#####

Stroll_ is a primitive C library that provides tools usefull to carry out common
C/C++ applicative tasks.

.. rubric:: Features

* common preprocessing macros,
* assertions,
* bit operations,
* bitmaps,
* length-value strings.

.. rubric:: Limitations

* GNU / Linux / Glibc_ userspace C only.
